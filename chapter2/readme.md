## variable

存储数据 var

- 变量是存储数据的地方
  变量包含三部分
  1. 变量名 未赋值时undefined
  2. value
  3. type
- 变量的类型由值来决定 
  弱类型(动态类型)语言 脚本语言 js php
  静态类型 c++ Java
- 表达式 多个变量进行操作 
计算和操作的优先级 2 + 3 * 6
字符串 模板  `${}`

- let const 
- 赋值 += ++ --
- scope 作用域 let 块级作用域{} 作用域之外 变量不可见

- 类型转换
  隐式转换 
    const a = 1;
    'string' + a
    'a' * 2 //NaN
  显示（强制）转换
    const a = 1;
    'string' + a
    Number('a') * 2 //2
